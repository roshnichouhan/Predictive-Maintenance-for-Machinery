## 📌 Project Overview

Predictive Maintenance is an AI-powered solution that analyzes machine sensor data and predicts possible equipment failures.

This project helps industries reduce unexpected downtime, improve maintenance scheduling, and increase machine efficiency using Machine Learning models.


## 🚀 Features

✅ End-to-End Machine Learning Pipeline  
✅ Data Cleaning & Preprocessing  
✅ Advanced Feature Engineering  
✅ Random Forest & XGBoost Models  
✅ Hyperparameter Tuning with GridSearchCV  
✅ Model Performance Evaluation  
✅ Feature Importance Analysis  
✅ SHAP Explainable AI  
✅ Streamlit Web Application for Prediction
